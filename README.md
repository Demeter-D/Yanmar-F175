# Yanmar F175 — Service Guide

A self-contained, mobile-friendly service guide for the Yanmar F175 compact
tractor (grey-import to the UK). Covers the full service schedule, step-by-step
procedures for filters, lubrication, adjustments, and inspections, plus UK
parts cross-references, torque values, and troubleshooting.

## Files

- `index.html` — single self-contained page. Open this in a browser.

That's all you need. React, Babel, and every procedure are bundled inside.

## Deploy to GitHub Pages

1. Upload `index.html` to the root of your repository (replace the existing
   one if you're updating).
2. Repo → **Settings → Pages**
   - **Source:** *Deploy from a branch*
   - **Branch:** `main` / `/ (root)`
   - Save.
3. Wait about a minute, then open
   `https://<your-username>.github.io/<repo-name>/` on any device.

## Open on iPhone

- Tap the share icon and **Add to Home Screen** for an app-style launch.
- The page works offline once Safari has cached it.
- The hamburger menu (top-right) opens the section drawer.

## Notes

- Yanmar part numbers were refined across the F175's 1984–1995 production
  run. Always verify against the engine and chassis plates before ordering.
- Service log entries persist in your browser's local storage; clearing
  Safari data will erase them.
