# Yanmar F175 — Service Guide

A self-contained, mobile-friendly service guide for the Yanmar F175 compact
tractor (grey-import to the UK). Covers the full service schedule, step-by-step
filter changes, UK parts cross-references, torque values, and troubleshooting.

## Files

- `index.html` — the page itself; open this in a browser
- `app.jsx` — shell, navigation, mobile drawer, tweaks
- `content.jsx` — all section content (specs, schedule, procedures, parts)
- `tweaks-panel.jsx` — the floating tweaks panel helper

All four files must sit in the same folder.

## Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `f175-service-guide`).
2. Upload all four files to the repository root.
3. Repo → **Settings → Pages**
   - **Source:** *Deploy from a branch*
   - **Branch:** `main` / `/ (root)`
   - Save.
4. Wait about a minute, then open
   `https://<your-username>.github.io/<repo-name>/` on your iPhone.

## Open on iPhone

- Tap the share icon and **Add to Home Screen** for an app-style launch.
- The page works offline once Safari has cached it.
- The hamburger menu (top-right) opens the section drawer.

## Notes

- Yanmar part numbers refined across the F175's 1984–1995 production run.
  Always verify against the engine and chassis plates before ordering.
- Service log entries persist in your browser's local storage; clearing
  Safari data will erase them.
